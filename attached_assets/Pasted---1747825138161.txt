| 観点              | 実装ポイント                                                                                                                                                                                                             | 補足                                               |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------ |
| **構造 (HTML)**   | \* `<header class="globalHeader">` 内に<br> `<h1>` ロゴリンク・`<nav>`・ハンバーガー`<button>` を配置。\*                                                                                                                             | デスクトップではロゴ＋横並びナビ、モバイルはハンバーガーでドロワーを開く構成。          |
| **レイアウト (CSS)** | \* `position: fixed; top:0; left:0; width:100%; z-index:1000;`<br>\* 高さ 64-72 px、左右 24 px パディング。<br>\* Flex で中央揃え (`display:flex; justify-content:space-between; align-items:center;`)                             | 上に常時固定し、ページを横スクロールしてもヘッダーは動かない。                  |
| **視認性**         | \* 背景：`background:rgba(255,255,255,.7); backdrop-filter:blur(8px);`<br>\* テキスト＆アイコン：`mix-blend-mode:difference; color:#fff;`                                                                                       | グラスモーフィズム＋ブレンドモードにより、下地が白でも黒でもロゴ／文字は必ずコントラストが付く。 |
| **インタラクション**    | 1. **スクロール連動ハイライト**<br> `gsap.registerPlugin(ScrollTrigger);` で横スクロール各セクションを監視し、該当ナビに `is-active` クラスを付替え。<br>2. **ハンバーガー**<br> `aria-controls` でドロワーをトグル、CSS で `transform:translateX(100%)` → `translateX(0)` 開閉。 | 横スクロール位置に応じてタブが自動で切り替わる。                         |
| **アクセシビリティ**    | \* `role="navigation"`, `aria-label="グローバルナビゲーション"`<br>\* ハンバーガーに `aria-expanded` を動的に付与                                                                                                                           |                                                  |
| **ビルド環境**       | - Nuxt 3 + Vite で静的書き出し（`.hash.js` が含まれる構成）。<br>- CSS は PostCSS で autoprefix・minify 済み、クラス名は BEM 風をベースにハッシュが付与。                                                                                                    | Mercari の他の LP と同じスタック。                          |
